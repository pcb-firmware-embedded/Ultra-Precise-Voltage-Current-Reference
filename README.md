# Ultra-Precise-Voltage-Current-Reference


This compact calibration module provides precise reference outputs for both voltage and current measurements. It generates stable levels of 10.000V, 1.0000V, 0.1000V, as well as 10.000mA, 1.000mA, and 100.0µA. The design is intended for quick and reliable verification of high-precision multimeters such as the Keysight 34461A. Careful component selection and layout ensure accuracy and repeatability under typical lab conditions. This makes it a convenient tool for routine calibration checks and measurement confidence.
